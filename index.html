YouTube channel RSS

Only have a handle? For example youtube.com/@kurzgesagt
- open the channel page,
- click … more links (under the channel header)
- scroll to the very bottom,
- open Share channel,
- then Copy channel ID,
- and paste that ID here.

How it works

The default channel RSS uses ?channel_id=UC… and follows all uploads. YouTube also exposes automatic playlists whose IDs are derived from the channel ID. For a 24-character channel ID starting with UC, replace that UC with one of the prefixes below, then request the feed with playlist_id instead of channel_id:

youtube.com/feeds/videos.xml?playlist_id=<PREFIX><rest of channel id after UC>
Prefix Meaning
UULatest: all public uploads (videos + Shorts + live)
UULFLatest: public long-form videos only (no Shorts)
UUSHLatest: Shorts only
UULVLatest: live streams only
UULPPopular: videos (~200, descending)
UUPSPopular: Shorts
UUPVPopular: live streams
UUMOLatest: members-only (all formats)
UUMFLatest: members-only videos
UUMSLatest: members-only Shorts
UUMVLatest: members-only live streams

These strings are not part of the official YouTube Data API contract. Member feeds only include items the signed-in account can access; RSS clients typically see nothing useful without membership.
